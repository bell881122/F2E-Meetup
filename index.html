講者／Daniel Wu

打造 Component 軍火庫

2019/04/25 四月號
more
講題簡介：

當自己寫越來越多前端，用了越來越多的 Open Source Library 之後，是不是有時候會遇到以下問題：
‧ 用的 Library 太大，明明只需要一個小功能，卻整個包山包海包進來。
‧ 怎麼修改現成的 Library 都沒辦法改成設計師畫出來的樣子。
‧ 嘗試硬改 Library 的內容到最後卻不記得裡面到底改過了什麼。
而後開始思考，我是不是能夠自己寫這些 Component 呢？ 又或者是在試著開發 Component 之後，發現怎麼寫都沒辦法符合需求、換個頁面就要重新修改一次，最後一個元件寫了五個版本放在不同的地方。 這次講者將分享自身在前端開發時因應 Style Guide 開發 Component 的經驗，以及開發時所考慮的方向，不論使用 Vanilla JS 或 Vue、 React 等框架，都能對想要自己寫 Component 甚至推出去成為 Open Source 可以有一些幫助。
